## YaMDb
### Описание
Проект YaMDb собирает отзывы пользователей на произведения. Произведения делятся на категории: «Книги», «Фильмы», «Музыка».
### Технологии
python 3.9.6  
django 3.0.5  
gunicorn 20.1.0
